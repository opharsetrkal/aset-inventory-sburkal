# Aset & Inventory SBU Reg Kalimantan
Web monitoring internal untuk Asset & Inventory SBU Regional Kalimantan.
Live: https://opharsetrkal.github.io/aset-inventory-sburkal

Stack
Google Sheets → Apps Script → GitHub Pages. Zero biaya, zero server.

Fase pengembangan
Fase 1 Administrasi Keuangan — monitoring billing & pembayaran pengiriman material ✅ 
Fase 2 Monitoring stok SAP gudang SBU Regional Kalimantan ✅
Fase 3 Live Monitoring Pengiriman STO — pergerakan material antar lokasi 
Fase 4 Planned Rekap Pengadaan — monitoring PO dari vendor/pusat 
Fase 5 Monitoring Penyerapan MB51 ✅
Fase 6 Monitoring Laporan PM ✅
Fase 7 Monitoring Kendala Doktek ✅
Fase 8 Monitoring SPR ✅
Fase 9 Monitoring SPP ✅

Cara update data
Input langsung di Google Sheets. Klik Refresh di dashboard untuk memuat data terbaru.
